## 0.1.0

- Initial release
- Inline search bar with hint text, clear button, and focus handling
- Expandable mode with animated expand/collapse
- Floating suggestions overlay with custom builder
- Inline filter dimensions (Airbnb-style Where/When/Who segments)
- Cross-platform adaptive behavior (mobile full-screen route, desktop floating overlay)
- Keyboard shortcuts for desktop (Escape to close, Tab to advance dimension)
- Full styling control via `ButterSearchBarStyle`
